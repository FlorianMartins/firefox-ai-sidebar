AI Sidebar — Réglages

100% BYOK : vos clés sont stockées localement dans ce navigateur (storage.local), jamais synchronisées, et envoyées uniquement à l'API que vous choisissez. Aucune clé n'est fournie par l'extension — renseignez la vôtre, ou pointez un modèle local sans clé.

⚡ Connexion rapide

Le plus simple : connecte ton compte OpenRouter (Google / GitHub / email) en 1 clic — la clé API est créée et enregistrée automatiquement ci-dessous, avec accès à des modèles gratuits.

Fournisseurs d'IA

Cliquez « 🔐 Se connecter » sur un fournisseur pour ouvrir sa page d'authentification, identifiez-vous avec votre compte, créez une clé API et collez-la. Beaucoup offrent un palier gratuit (Gemini, Groq, Mistral, Cerebras, OpenRouter). OpenRouter propose en plus une vraie connexion par compte (OAuth) qui débloque tous les modèles. Vous pouvez aussi activer un modèle local (Ollama / LM Studio). Un fournisseur n'apparaît dans le sélecteur qu'une fois connecté.

🤖 Mode agent

En mode Agent, l'IA pilote le navigateur via des outils (ouvrir un onglet, cliquer, remplir, lire la page…). Beaucoup de modèles rapides/gratuits (ex. Llama) ne savent pas appeler d'outils et le mode agent reste alors inactif. Choisissez ici un modèle capable d'appeler des outils (Claude, GPT-4o, DeepSeek, Gemini…). Laissez Auto pour utiliser le modèle sélectionné dans la sidebar.

🌐 Recherche web

En mode Web, la requête est envoyée à un modèle capable de chercher sur le web — de préférence Perplexity (Sonar) ou un modèle gratuit OpenRouter (plugin web intégré), pour éviter de dépenser sur Claude. Laissez Auto pour un choix automatique parmi vos fournisseurs connectés.

Génération d'images

Utilise l'endpoint compatible OpenAI /images/generations. Les tailles dépendent du modèle (gpt-image-1 : 1024²/1536×1024/1024×1536 · DALL·E 3 : 1024²/1792×1024/1024×1792 · DALL·E 2 : 256²/512²/1024²). Le 4K/1440p natif n'est pas produit par ces modèles — générez en HD puis agrandissez.

Sécurité & confidentialité

Zéro rétention serveur : cette extension n'a aucun serveur. Vos données (clés, conversations, contenu des pages) restent dans ce navigateur et ne sont envoyées qu'à l'API IA que vous choisissez. Aucune analytique, aucune télémétrie.
